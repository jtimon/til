* til

A programming language with ownership semantics, type inference, and compile-time
garbage collection. This is the second implementation attempt, written in C. The
first attempt in Rust lives at https://github.com/jtimon/rstil.

** Build

#+begin_src sh
make
#+end_src

** Run

Interpret a program:
#+begin_src sh
bin/ctil interpret program.til
#+end_src

Compile to C and run:
#+begin_src sh
bin/ctil run program.til
#+end_src

** Test

#+begin_src sh
make test
#+end_src
