** #2 ref keyword: non-owning references for returns, declarations, and struct fields
  ref returns: ~returns ref I64~ — caller does not own, ASAP skips.
  ref declarations: ~ref x := min(a, b)~ — binds without ownership.
  ASAP lifetime extension for args to ref-returning calls.
  ref struct fields: store pointer, clone copies pointer, delete skips.
  null literal: only valid in ref positions (ref locals, ref struct fields,
  borrowed params). Rejected for own/shallow params and non-ref locals.
  Args are already borrowed (ref) by default.
