** #89 Revert to monolithic compilation (remove per-module file splitting)
  Removed per-module .c/.h emission from builder.c ([[./74.org][#74]]). All struct/enum
  defs, function declarations, and function bodies now go into a single .c
  with forward declarations inlined. The ~mode lib~ .h (~modes.h~) is still
  generated separately by ~build_header~.

  ~bootstrap/~ dropped from 70 files to 6: ~modes.c~, ~modes.h~,
  ~modes_forward.h~, ~til.c~, ~til.h~, ~til_forward.h~.

  ~self_diff~ was already removed in 5d365ff18. ~compile_lib~ and
  per-library scavenging remain (separate concern).
