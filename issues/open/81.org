** OPEN #81 UPtr type alias for pointer-sized integers

:DEPENDENCIES: [[./79.org][#79]] [[./25.org][#25]]
  ~UPtr := U64~ on 64-bit platforms, ~UPtr := U32~ on 32-bit.
  Platform-determined, NOT user-configurable (unlike USize [[./80.org][#80]]).

  This is the type for raw pointer arithmetic, address calculations,
  and anything that must match the platform's native pointer width.
  Distinct from USize which is for container sizes and can be
  configured independently (e.g. USize := U32 on a 64-bit platform
  is fine — you rarely need 4 billion element containers).

  Post-self-hosting — depends on [[./25.org][#25]] (multi-platform support) to
  actually have different platforms to target. On Linux x86_64 only,
  UPtr is just U64.
