** OPEN #151 TempleOS target

:DEPENDENCIES: [[../fixed/12.org][#12]]

  Add a TempleOS target. This is not part of the hosted 32-bit target
  work anymore: TempleOS is a separate toolchain, runtime, filesystem,
  graphics, and testing environment.

  The old broad 32-bit scope was split:

  - Hosted 32-bit Linux/Windows targets are tracked in [[./232.org][#232]].
  - Browser WASM is tracked in [[./228.org][#228]].

*** Scope

  - Emit or translate til programs to HolyC or another TempleOS-native
    representation.
  - Replace POSIX/Windows runtime assumptions with TempleOS primitives.
  - Implement ~mode gui~ on TempleOS using native graphics rather than
    raylib.
  - Define a QEMU-based test loop using a TempleOS VM image.

*** Notes

  TempleOS has no normal C compiler path comparable to the Linux,
  Windows, or browser targets. Treat it as deferred platform work, not
  as one more clang cross target.
