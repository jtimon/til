** OPEN #42 til install: system-wide library installation

~til install mylib.til~ — build a mode lib/pure file and install
  artifacts to system paths for use by other programs:
  - ~.so~ / ~.a~ → system lib path (e.g. ~/usr/local/lib~)
  - ~.h~ → system include path (e.g. ~/usr/local/include~)
  - ~.til~ → til lib path (TBD — e.g. ~~/.til/lib/~ or similar)

  Depends on [[./37.org][#37]] (mode lib) which already produces .c, .h, .til, .so, .a
  via translate/build. This issue adds the install step that copies those
  artifacts to system-wide locations so they can be used without path
  manipulation.
