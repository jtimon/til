1. Read the next issue number from `issues/NEXT_ISSUE.txt`.
2. Create a new file in issues/open/ with that number, e.g. issues/open/189.org
3. First line: "# priority: high", "# priority: medium", or "# priority: low"
4. Then: "** OPEN #189 Title" followed by the issue body
5. After creating an issue, increment `issues/NEXT_ISSUE.txt` manually.
6. To start work: change OPEN -> WIP in the title line (no directory move needed)
7. When fixed:  git mv issues/open/NNN.org issues/fixed/NNN.org
8. When cancelled: git mv issues/open/NNN.org issues/cancelled/NNN.org
9. Run: make test  (regenerates issues/summary.org; increment `issues/NEXT_ISSUE.txt` manually)
