** Blockers

*** BLOCKER A: Function pointer dispatch (dispatch_init + ext_dispatch_builtin)
dispatch_init stores C function pointers in dispatch_map.
ext_dispatch_builtin retrieves and calls them via cast.
til cannot take function addresses or call Dynamic as function.

*** BLOCKER B: ext_dispatch_ffi
VLAs (void *args[nargs]), union for return buffer, ffi_call/FFI_FN/FFI_DEFAULT_ABI macros.
