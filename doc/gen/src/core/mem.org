* swap_bytes
:PROPERTIES:
:CUSTOM_ID: til.src.core.mem.swap_bytes
:group: Core
:END:

Compiler-internal raw byte swap: the desugarer lowers `param = rhs`
for mut aggregate params to `swap_bytes(param, temp, T.size())`
(hoist_param_swap_assign), which runs after generic instantiation and
so cannot call the typed swap below. Not for user code -- it has no
ownership semantics by itself (the lowering pairs it with the temp's
scheduled delete). Dies with #302 stage 4's first-class move marker.

func swap_bytes(a: Dynamic, b: Dynamic, shallow size: UPtr)
