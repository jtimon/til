## The #39 module swap pattern (document in self.org)

For each C module → til replacement:
1. Write the .til file (mode lib)
2. Make .til import the previous "top" module
3. Update til_core: translate the new top instead of (or in addition
   to) the old one. Copy generated .c/.h to bootstrap/.
4. Update Makefile SRCS/HDRS: bootstrap/new.c replaces bootstrap/old.c
5. Update C module includes: bootstrap/new.h replaces old headers
6. Remove old C source (src/c/old.c, src/c/old.h or modes.h)
7. Remove functions from ext.c that are now in the .til module
8. `make clean && make test` at EACH step
