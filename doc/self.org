* Self-hosting build

  See Makefile header comments and ~make help~ for the full reference.

  Key concepts:
  - ~bin/til_boot~: built from last commit via git. Always works.
  - ~bin/til~: built by til_boot from current .til sources.
  - ~boot/~: generated C, committed for til_boot. Regenerated every build.
  - ~make test~ does everything. Commit includes boot/.
  - ~make clean && make test~ always works.
  - ~make revert_boot~ for recovery.
